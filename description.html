Three.js Crossy Roads

Marilyn Fang and Kate MacVicar

Description

We used Three.js to create a crossy roads-like game where players are a duck that must navigate through a jungle of trees and crocodiles to get home!

List of features in this project:

  • Modeling
  • Material, Lighting, and Shading
  • Changing Camera Position
  • User Interaction
  • Animation
  • Texture Mapping
  • Sounds
